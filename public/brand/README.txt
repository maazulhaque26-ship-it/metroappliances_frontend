METRO APPLIANCES — BRAND ASSETS
================================

Place the following file in this directory:

  logo.png   — The official Metro Appliances circular badge logo (PNG with white/transparent background)

This file is referenced throughout the app as /brand/logo.png
It is also used as the favicon in all browsers (Chrome, Safari, Firefox, Edge, mobile).

DO NOT rename the file.
DO NOT move it to another folder.
DO NOT configure it in any admin panel or database.
